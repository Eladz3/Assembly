Lab 4: Sorting Floats

-----------
DESCRIPTION

This program receives three IEEE strings, converts them into numeric values, sorts them in ascending order then
prints them out as IEEE numbers and decimal numbers.

-----------
FILES

-
Lab4.asm

This is the pseudo code and assembler code that runs the program.

-
Diagram.pdf

This is a flow chart diagram that explains what is happening in the .asm file.

-----------
INSTRUCTIONS

This program is run on Mars

I.   Once the user runs the Lab4.asm file, all they have to do is enter three hex values with a space in between
     (Exp: "0x00000000 0x00000001 0x00000002")

II.  Run the program and the original values, sorted values, and sorted decimal values should print.
